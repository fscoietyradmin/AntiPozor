AntiPozor v3.0 — официальная документация

AntiPozor — это многофункциональный плагин для серверов Minecraft, работающих на Paper 1.21.11 и любых других форках, поддерживающих API версии 1.21 (Spigot, Purpur, Airplane и др.). Плагин объединяет в себе мощный античит, защиту от DDoS-атак, систему наказаний с красивыми эффектами и автоматическое обновление через GitHub. Разработан для обеспечения безопасности и порядка на сервере.
✨ Основные возможности
🛡️ Античит (11 проверок)

Плагин автоматически отслеживает подозрительное поведение игроков и применяет наказания при превышении порогов нарушений. Все проверки работают на событиях и не нагружают сервер.
Проверка	Описание
SpeedCheck	Обнаружение ускорения (speed-hack)
FlyCheck	Обнаружение полёта без прав
ReachCheck	Обнаружение атаки на слишком большом расстоянии
KillAuraCheck	Обнаружение автоматических ударов (KillAura)
AutoClickCheck	Обнаружение авто-клика (CPS выше нормы)
NoFallCheck	Обнаружение игнорирования урона от падения
FastBreakCheck	Обнаружение слишком быстрого разрушения блоков
PacketRateCheck	Обнаружение подозрительно высокой частоты пакетов
TimerCheck	Обнаружение ускорения игрового времени
AntiVapeCheck	Обнаружение характерных паттернов Vape-клиентов
AntiXRay	Базовое обнаружение X-Ray (подозрительная добыча руд)

Каждая проверка имеет настраиваемый порог нарушений в config.yml. При достижении порога игрок автоматически банится с красивой анимацией.
🎨 Красивые наказания

    BeautifulBan – при бане игрок поднимается в воздух, вокруг появляются частицы огня, звучит взрыв фейерверка, а всем игрокам отправляется красочное сообщение.

    Кастомизируемые сообщения – все сообщения (бан, кик, мут, предупреждение) настраиваются в конфиге с поддержкой цветов (&4, &6 и т.д.) и переменной %reason%.

📋 Система наказаний

    Warn – выдача предупреждений; при накоплении 3 варнов игрок автоматически получает мут на 1 час, при 5 варнах – красивый бан.

    Mute / Unmute – временная блокировка чата с указанием причины и длительности.

    TempBan – временный бан (команда /antipozor tempban).

    Kick – кик с красивым сообщением.

    History – просмотр истории наказаний игрока (базовая версия).

🌐 Защита от DDoS

    Отслеживание количества подключений с одного IP за заданный интервал.

    Автоматическая временная блокировка IP при превышении порога.

    Белый и чёрный списки IP для ручного управления.

    При обнаружении массовой атаки выводится предупреждение в консоль и в чат администраторам.

🔄 Автоматическое обновление через GitHub

    Плагин при запуске проверяет последний релиз в указанном GitHub-репозитории.

    При наличии новой версии администраторы получают уведомление в игре и консоли.

📦 Управление IP-списками

    Команды для добавления/удаления IP в белый (доверенные) и чёрный (всегда заблокированные) списки.

    Удобный tab-complete для всех команд.

🔐 Разрешения (Permissions)

    antipozor.admin – полный доступ ко всем командам и функциям плагина (по умолчанию выдаётся операторам).

    antipozor.bypass – игнорирование античит-проверок и DDoS-блокировок (для доверенных игроков).

⚙️ Конфигурация (config.yml)

Все настройки плагина вынесены в удобный YAML-файл, который автоматически создаётся при первом запуске. Вы можете редактировать его в любое время и перезагружать командой /antipozor reload.
Основные секции:
yaml

# Точные ники для бана
exact-names:
  - "kotnk_xd"
  - "rom7737"
  - "CRASH_RADMIN"
  - "ogurec"
  - "frested"
  - "printin"
  - "RadminHack"

# Подстроки для бана (бан, если ник содержит одну из этих строк)
blocked-substrings:
  - "hack"
  - "crash"

# Красивые сообщения (используй & для цветов)
messages:
  ban: "&4&lТЫ ЗАБАНЕН! &cПричина: %reason%"
  kick: "&e&lТЫ КИКНУТ! &eПричина: %reason%"
  mute: "&7[&cМут&7] &fВы замьючены. Причина: %reason%"
  warn: "&6[&eПредупреждение&6] &f%reason%"
  beautiful-ban: "&4&l⚡ &cИгрок %player% был забанен за &e%reason% &4&l⚡"

# Настройки DDoS-защиты
ddos:
  alert-threshold: 10          # количество попыток входа для блокировки
  time-window: 10000            # интервал в миллисекундах (10 сек)
  block-duration: 60000         # время блокировки IP (60 сек)

# Белый и чёрный списки IP (можно редактировать через команды)
whitelist-ips: []
blacklist-ips: []

# Настройки античита (пороги срабатывания)
anticheat:
  speed:       { ban-threshold: 10 }
  fly:         { ban-threshold: 5 }
  reach:       { ban-threshold: 10 }
  killaura:    { ban-threshold: 10 }
  autoclick:   { ban-threshold: 20 }
  nofall:      { ban-threshold: 5 }
  fastbreak:   { ban-threshold: 5 }
  packetrate:  { ban-threshold: 100 }
  timer:       { ban-threshold: 10 }
  antivape:    { ban-threshold: 10 }
  antiray:     { ban-threshold: 10 }

# Проверка обновлений (укажи свой репозиторий)
update-checker:
  enabled: true
  github-repo: "твой-логин/AntiPozor"
  check-interval: 43200         # проверять каждые 12 часов (в секундах)

📥 Установка и требования
Требования

    Версия Minecraft: 1.21 или выше (рекомендуется Paper 1.21.11 или любой форк, поддерживающий Paper API 1.21).

    Java: 21 или выше.

    Зависимости: Нет, плагин самодостаточен.

Установка

    Скачайте последнюю версию плагина (JAR-файл) с GitHub Releases.

    Поместите файл в папку plugins вашего сервера.

    Перезапустите сервер (или выполните /reload, но рекомендуется полный рестарт).

    При необходимости отредактируйте файл plugins/AntiPozor/config.yml.

    Выдайте себе права antipozor.admin, если не являетесь оператором (обычно операторы уже имеют доступ).

Совместимость

Плагин разработан с использованием Paper API 1.21 и должен корректно работать на всех серверных ядрах, основанных на Paper/Spigot версии 1.21 и выше, включая:

    Paper 1.21.11

    Spigot 1.21.11

    Purpur 1.21.11

    Airplane, Pufferfish, Tuinity и другие форки.
